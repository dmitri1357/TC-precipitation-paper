This repository contains code for the manuscript titled "Contribution of Tropical Cyclones to Hourly Precipitation Extremes in the Contiguous United States" by Kalashnikov et al. (2026). The associated datasets which this code uses are stored in the following open-access repository: https://doi.org/10.5281/zenodo.18853288.
